## How to Add Your Name

This is your first Git exercise! Here's how:

1. **Open this file** (`STUDENT_ROSTER.md`) in VS Code
2. **Add your line** following the format above
3. **Save the file** (Ctrl+S or Cmd+S)
4. **Stage your changes:**
   ```bash
   git add STUDENT_ROSTER.md
   ```
5. **Commit with a message:**
   ```bash
   git commit -m "Add [Your Name] to student roster"
   ```
6. **Push to GitHub:**
   ```bash
   git push origin main
   ```
7. **Verify on GitHub** - go to your fork and see your name!

**Stuck?** Ask in the team channel or check the [Setup Guide](docs/SETUP.md).
