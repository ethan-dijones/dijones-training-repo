# Week 5-6: Integration & Final Projects

Welcome to the final weeks! You'll combine everything you've learned to build complete applications.

## 🎯 Final Weeks Goals

By the end of Week 6, you will:
- ✅ Work with APIs
- ✅ Integrate frontend and backend
- ✅ Build a complete application
- ✅ Present your final project

## 📚 Prerequisites

- Completed Weeks 1-4
- Comfortable with HTML, CSS, JavaScript, and Python
- Ready to build something real!

## 🎓 Final Project Options

Choose one to build:

### Option 1: IT Asset Tracker
Track computers, monitors, and software licenses with:
- Add/edit/delete assets
- Search and filter
- Generate inventory reports

### Option 2: Knowledge Base System
Searchable FAQ/article system with:
- Add/edit articles
- Category filtering
- Search functionality

### Option 3: Incident Timeline Tool
Track and visualize incidents with:
- Add incident details
- Timeline display
- Export reports

## 📂 Project Structure

Create your project in: `exercises/week5-6/final-project/`

```
final-project/
├── README.md           # Project documentation
├── index.html          # Main page
├── style.css           # Styling
├── script.js           # Frontend logic
├── data.json          # Data storage
└── generate_data.py   # Backend script (optional)
```

## 📋 Project Requirements

- [ ] Frontend (HTML/CSS/JavaScript)

- [ ] Data persistence (JSON files)
- [ ] Professional appearance
- [ ] Works without errors
- [ ] Code is commented
